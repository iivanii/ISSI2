Organización de los distintos módulos que componen una aplicación

### Built-in-architectures
- Por capas
	Los módulos de una capa ofrecen una funcionalidad común. Solo pueden interaccionar con capas adyacentes

- Cliente-servidor
	Divide la funcionalidad entre servidor y cliente

- Componentes
	Cada módulo se encarga de una funcionalidad a la que se accede a través de una interfaz

- Centrada en datos
	El elemento principal es una base de datos, a la que solo accede un módulo concreto

- P2P
	Divide las tareas de forma equitativa entre distintos nodos comunicándose entre sí

- Microservicios
	Las apps son colecciones de servicios levemente acoplados entre sí
