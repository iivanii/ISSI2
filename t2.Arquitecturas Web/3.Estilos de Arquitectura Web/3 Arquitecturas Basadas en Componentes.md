

--------------------------------------------------------------------------
Aplicación estructurada en componentes (funcionalidades relacionadas) cuyos componentes son reutilizables y están intercomunicados por interfaces 

- **Ventajas**: Cada componente es más fácil de mantener y probar
- **Desventajas**: Esfuerzo de diseño previo para definir y separar componentes
