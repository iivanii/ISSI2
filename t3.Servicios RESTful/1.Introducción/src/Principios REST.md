
------------------------------------------------------------

- Recurso: Cualquier cosa que expongamos en La Web
- URL (Uniform Resource Identifier): Identifica de manera única a un recurso web y, al mismo tiempo, lo hace direccionable y manipulable utilizando un protocolo de aplicación HTTP
- Un recurso puede tener varias [[4 URL|URL]]
- Un recurso puede tener varias representaciones
- Un recurso puede tener enlaces a otros recursos
