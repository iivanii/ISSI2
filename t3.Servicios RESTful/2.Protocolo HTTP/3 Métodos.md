
--------------------------------------------------------------------------

- **GET**
	**Obtiene** una representación del recurso especificado
 ^de3864
- **POST**
	**Envía** datos al servidor para su procesamiento. Puede resultar en la creación de un nuevo recurso
 ^2a9221
- **PUT**
	**Actualiza totalmente** el recurso especificado
 ^7685a6
- **PATCH**
	**Actualiza parcialmente** el recurso especificado

- **DELETE**
	Borra el recurso especificado ^c5db9d

- **HEAD** 
	Pide una respuesta idéntica a la que correspondería a una petición **GET**, pero sin el cuerpo de la respuesta. Útil para comprobar si la caché sigue siendo válida mediante la cabecera _Last-Modified_
 ^9baa9d

- **OPTIONS**
	Devuelve los métodos HTTP que el servidor soporta para una [[4 URL|URL]]
