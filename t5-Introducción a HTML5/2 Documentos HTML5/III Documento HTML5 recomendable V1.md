```HTML
<!DOCTYPE html>
<html>
	<head>
		<title>Documento HTML5 
recomendado</title>
	</head>
	<body>
		<p>Otro documento mejor 
estructurado</p>
</body>
</html>
```


- \<html>
	Elemento raíz del documento

- \<head>
	Cabecera del documento; aparte del título, puede contener otra información sobre el documento

- \<body>
	Cuerpo del documento

- lang=“es”
	India el idioma que presenta el contenido, útil para buscadores y lectores de pantalla

- \<meta charset="utf-8">
	Indica codificación del fichero

- \<link href="estilos.css" rel="stylesheet”>
	Hoja de estilo asociada

- \<script src="scripts.js"> \</script>
	Fichero importado con código JavaScript
